Задание:
Для интерфейса необходимо определить 1 свойство и 2 метода. Абстрактный класс должен содержать 3 - 5 свойств и 3 - 5 методов (включая унаследованные свойства интерфейса). Класс должен содержать дополнительно 2 свойства и 2 метода. В программе реализовать работу со списком объектов
interface - Производство
abstract class - Завод
class - Кирпичный завод
